IMPLEMENT MEDIA CENTER, NEWS, RESOURCES AND ANNOUNCEMENTS MODULE

Create a complete Content Management System (CMS) for PesaMatrix that allows Admins to create, edit and delete content from the Admin Dashboard.

GENERAL REQUIREMENTS

Only Admin users can:

- Create
- Upload
- Edit
- Publish
- Unpublish
- Delete

Regular users can only view published content.

All uploads must support mobile phone gallery uploads.
